Welcome to AWN

AWN, or the AsianWordNet Project, is the result of the collaborative effort in creating an interconnected WordNet for Asian languages. AWN provides a free and public platform for building and sharing among Asian WordNets. The distributed database system and user-friendly tools have been developed for user. AWN is easy to build and share. Let’s build AWN together for the connection of us.

In Asia, there are some efforts in developing WordNets of some languages. But the number of languages that has been successfully developed their WordNets is still limited to some active research in this area. However, the extensive development of WordNet in other languages is important, not only to help in implementing NLP applications in each language but also in inter-linking WordNets of different languages to develop multi-lingual application to overcome the language barrier.



To achieve the AWN goal, we provide a communication platform for finishing WordNet for Asian languages. AWN builds Asian language WordNet based on the English Princeton WordNet. Therefore, the original structural information is inherited to the target WordNet through its translation and sense ID. AWN also connects each WordNet to build the complete Asian WordNet via the Princeton English WordNet.

WNMS has been developed to facilitate the process of the connection between the members, the database storages and the English WordNet translation. Tools in WNMS allow the members to create their local WordNets, to visualize the result of WordNet structure and to use the AWN dictionary to search across languages.
